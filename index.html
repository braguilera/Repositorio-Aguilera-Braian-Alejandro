Proyectos

Proyectos

  • Gestor consorcio

    Proyecto grupal universitario con enfoque en gestión de consorcios. Diseñé y desarrollé el frontend usando React, con un diseño UX optimizado y animaciones interactivas. Implementé el backend con Java y SQL Server, resolviendo el desafío de eliminación en cascada en la base de datos. La aplicación incluye un sistema de roles (inquilinos, dueños, empleados) con acceso específico mediante ruteo dinámico y presentación de datos tipo tabla usando React y CSS.

  • Portafolio Dot Dager

    Landing page creada en un concurso de tres días para un creador de contenido. Fui responsable de representar su humor, estilo y personalidad a través de un diseño interactivo, con animaciones fluidas en secciones clave. Incluye enlaces a sus redes sociales, una galería de videos y una sección de su juego favorito. Utilicé HTML, CSS y JavaScript, logrando un diseño que se destacó en el ranking del concurso.

  • Adivina el número

    Juego lógico realizado en React y CSS, donde el objetivo es adivinar un número secreto en 10 intentos. El juego utiliza un teclado interactivo para ingresar combinaciones y proporciona pistas sobre los números en la posición correcta, incorrecta o que forman parte del número. Con niveles de dificultad jugando con o sin números repetidos, este proyecto me permitió aprender sobre animaciones, manejo de datos entre componentes y lógica condicional en React.
